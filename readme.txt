== Description ==

Anim8 is a plugin for wordpress that allows you to create a slider with an unlimited number of slides. It's designed to be lightweight and easy to use without a ton of options. 
However I plan to add many more features in the future.



== Installation ==

1. Upload `Anim8.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress




== Frequently Asked Questions ==

= Displaying the Slider =

Simply write [Anim8] into your post/page editor where you want the slider to appear.

= What about putting it in my theme? =

<?php echo do_shortcode('[Anim8]'); ?>
